# PetroCorp - Proyecto Final Unidad 1

Sitio web profesional para una empresa de hidrocarburos, diseñado por **Jason Muñoz**.

## Contenido del Proyecto
- `index.html`: Página principal con carrusel panorámico y secciones de historia/misión.
- `pagina.html`: Sección de productos con lista de precios y formulario corporativo.
- `CSS/`: Estilos personalizados (`iniciostyle.css` y `paginastyle.css`).
- `imagenes/`: Recursos visuales organizados por página.
- `Documento_de_Análisis.md`: Explicación clara sobre el desarrollo.

## Tecnologías
- HTML5
- Bootstrap 5 (Responsivo)
- CSS3 con variables personalizadas
- JavaScript (Funcionalidades básicas)
