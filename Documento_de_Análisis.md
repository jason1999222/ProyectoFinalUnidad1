# Documento de Análisis del Proyecto - PetroCorp
 
## Descripción del Sitio
 
Este es un proyecto de una página web para una empresa de hidrocarburos. La idea principal fue crear un sitio que se vea serio y profesional pero usando herramientas sencillas de diseño web que vimos en clase.
 
El sitio tiene dos secciones principales:
- **Página de Inicio:** Tiene un carrusel de imágenes grandes que se mueven solas para mostrar la empresa. También tiene secciones con texto e imágenes que explican la historia y la misión de PetroCorp.
- **Página de Productos:** Aquí se muestra lo que la empresa vende, como gasolina y diesel, con sus precios en pesos colombianos. También hay un formulario largo para que los clientes dejen sus datos y su NIT si quieren que los contacten.
 
## Organización de los Archivos
 
Para que el proyecto esté ordenado, se usó esta estructura:
- Una carpeta **CSS** para los archivos de diseño (`iniciostyle.css` y `paginastyle.css`).
- Una carpeta **Imagenes** con subcarpetas para cada página (`pagina1` y `pagina2`), para que no se mezclen las fotos.
- Los archivos **index.html** y **pagina.html** están en la carpeta principal para que sea fácil abrirlos.
 
## Herramientas Usadas
 
- **HTML:** Para armar toda la base de la página con etiquetas como section, footer y nav.
- **Bootstrap 5:** Se usó para que la página sea "responsiva", o sea, que se vea bien en celulares y en computadoras sin que se descuadre nada.
- **CSS:** Para poner los colores gris y dorado, y para que los botones y el carrusel se vean mejor.
- **JavaScript:** Un archivo sencillo para que el sitio tenga algo de movimiento y los formularios funcionen.
 
## Conclusión
 
El proyecto cumple con mostrar la información de la industria del petróleo de forma clara. Se aprendió a organizar mejor las carpetas de un proyecto y a usar componentes de Bootstrap para que el diseño sea más rápido de hacer.
